# README #

This repository will contain all the files related to the use of bitwise manipulation, making use of operator such as &(AND), |(OR), <<(LEFT SHIFT OPERATOR),
>>(RIGHT SHIFT OPERATOR) and XOR operator, such files are:

* 0-binary_to_uint.c
* 1-print_binary.c
* 2-get_bit.c
* 3-set_bit.c
* 4-clear_bit.c
* 5-flip_bits.c
